# justore
Justore is a very simple personal file server written in modern C++. It embodies the UNIX philosophy and does one thing at once - acting as your file server, no sharing, no other users, no calendar.

This project is inspired from [copyparty](https://github.com/9001/copyparty). Just like copyparty, Justore is designed to run smoothly on a 1C1G Server without much usage of resources.

## Important Notice
This project is still working in progress. There may not be any code in the repository.
